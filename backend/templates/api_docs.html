Page Number Specification

If the pages parameter is included, please provide the page numbers in the following format:

  • Page numbers start from 1. For example, the first page is 1.
  • To specify multiple pages, separate the numbers with commas. For example, to select pages 1, 3, and 5, use 1, 3, 5.
  • To specify a range of pages, use a hyphen (-). For example, to select pages 1 to 5, use 1-5.
  • Spaces are allowed between numbers and separators. For example, 1, 3 , 5 - 10 is valid.
  • The page numbers will be automatically sorted during processing. Therefore, the order in which you specify the page numbers does not matter. For example, specifying 5, 3, 1 will automatically be sorted to 1, 3, 5 during processing.
